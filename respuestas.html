¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

HTML significa Lenguaje de Marcado de Hipertexto. es un lenguaje de marcado que consiste en una serie de elementos
que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. etiquetas:

<p>

<h1>

<h2>

<h3>

<h4>

<h5>

<h6>

<head>

¿Cómo es la anatomía de un elemento HTML? Dar un ejemplo con la etiqueta <p>

tiene 4 partes principales:

La etiqueta de apertura: consiste en el nombre del elemento, encerrado por los signos de mayor y menor (<>)

La etiqueta de cierre: es igual que la etiqueta de apertura, pero tiene la barra de cierre de por medio (</>)

El contenido: es el contenido del elemento

El elemento: es el conjunto entre: la etiqueta de apertura, más la etiqueta de cierre, más el contenido

ejemplo: <p>textolargo</p>

¿Cuál es la anatomía básica del HTML?

<!DOCTYPE html>(encabezado)
<html lang="en"> (lenguaje)
<head> (encabezado)
<meta charset="UTF-8"> (configuracion)
<meta http-equiv="X-UA-Compatible" content="IE=edge"> (configuracion)
<meta name="viewport" content="width=device-width, initial-scale=1.0"> (configuracion)
<title>Document</title> (nombre)
</head> (fin de encabezado)
<body> (cuerpo)

</body> (fin de cuerpo)
</html> (fin del html)

¿Qué es y que función cumple el CSS?

el CSS es un lenguaje informático, que puede controlar con precisión cómo se presentan
los documentos a los usuarios: cómo se diseñan y compaginan

¿Cómo se incorpora el CSS dentro del HTML?

El CSS se puede usar para estilos de texto muy básicos como, por ejemplo, cambiar el color y el tamaño de los encabezados y los enlaces.
Se puede utilizar para crear un diseño

¿Cual es la anatomia de una regla CSS?

h1 {
color: red;
font-size: 5em;
}

¿Cuales son los tipos de sectores que puede utilizar CSS?

los selectores definen sobre que elementos se aplicaran un conjunto de reglas CSS
selectores basicos: selector de tipo, selector de clase selector de ID, selector universal, selector de atributo
selectores combinadores: de hermanos adyacentes, general de hermanos, de hijo, de descendientes de columna
selectores pseudoclases: selecciona elemento mediante informacion fuera de documentos
